Projekt EKG pomocou ESP32, AD8232 a konektoru na tento komponent. 
Pomocou programu v Arduino IDE ESP32 zasiela cez určený USB-A port do PYTHON programu dáta, ktorý následne vykresluje graf srdcovej funkcie a zobrazuje aj tep. 
Obrázky 12255 a 12257 zobrazujú zapojenie.
Obrázky Figure_1 a Figure_2 zobrázujú príklad vykreslenie srdcovej funkcie pomocou tohto projektu.
